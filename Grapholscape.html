# setRenderer(rendererKey, keepViewportStateopt)

Change the rendering mode.

Parameters:
Name Type Attributes Default Description
rendererKey string

the rendering/simplifation mode to activate: default, lite, or float
keepViewportState boolean <optional>
 true

if false, viewport will fit on diagram. Set it true if you don't want the viewport state to change. In case of no diagram displayed yet, it will be forced to false. Default: true.

Note: in case of activation or deactivation of the float mode, this value will be ignored.
